Concepts, tools, and what each program does

This page explains the basic concepts and the external programs used by the pipeline. Every tool below includes a link to its official webpage so you can download it.

Follow the workflow Back to home

Key concepts (plain language)

FASTA file

A FASTA file is a plain-text file that stores sequences. Each sequence starts with a header line that begins with >.

Multiple sequence alignment

An alignment lines up sequences so you can compare the same positions across samples. This is needed before building a phylogenetic tree.

Trimming

Trimming removes columns in the alignment that are mostly gaps or noisy. This can make the tree more stable and easier to interpret.

Phylogenetic tree

A phylogenetic tree groups sequences by similarity. Sequences that are more similar tend to appear closer together.

Programs used by this project

Alignment

MUSCLE

Builds a multiple sequence alignment from your input FASTA. This pipeline uses MUSCLE first to create *_aligned.fasta.

Official MUSCLE page

Trimming

ClipKIT

Removes low-information columns from the alignment. The default mode in this workspace is smart-gap.

Official ClipKIT page

Tree inference

IQ-TREE 2

Infers a phylogenetic tree from the trimmed alignment. In this workspace it is invoked with model testing and ultrafast bootstrap.

Official IQ-TREE page

Useful notes for this repository

  • The main pipeline script is code/aln_trim_tree.py.
  • MUSCLE is vendored for Windows in this repo as code/muscle-win64.v5.3.exe. On macOS/Linux, download MUSCLE and pass --muscle-exe if needed.
  • IQ-TREE is expected at code/bin/iqtree2.exe by default. On macOS/Linux, download IQ-TREE 2 and pass --iqtree-exe.
  • ClipKIT must be available as clipkit (or provide --clipkit-exe).

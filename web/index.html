blaZ phylogeny project

A small, reproducible workspace to build and visualize a blaZ phylogenetic tree

Collect and filter sequences, build an alignment, trim it, infer a tree, and visualize the results. This site is written for non-technical users: clear steps, expected files, and where to find outputs.

What this repository contains

Scripts and binaries

code/

The main automation entrypoint is code/aln_trim_tree.py (MUSCLE → ClipKIT → IQ-TREE). There is also Fasta to Excel.py to summarize FASTA headers into an Excel file.

Results

iqtree_results/

IQ-TREE outputs: tree files (*.treefile), logs (*.log), and run metadata (*.iqtree).

Visualization

R_blaZ_tree/

R scripts to render and annotate the tree using ggtree. Tip groups can be defined and colored for interpretation.

Pipeline summary (README)

code/aln_trim_tree.py runs three external tools in order: MUSCLE (alignment) → ClipKIT (trimming) → IQ-TREE (tree inference).

Requirements

  • Python 3
  • MUSCLE v5 executable (this repo includes code/muscle-win64.v5.3.exe)
  • ClipKIT available as clipkit in your PATH (or provide --clipkit-exe)
  • IQ-TREE 2 executable (this repo expects code/bin/iqtree2.exe by default)

What it creates

  • <input>_aligned.fasta
  • <input>_aligned_trimmed.fasta
  • IQ-TREE outputs under iqtree_results/<input_basename>.*

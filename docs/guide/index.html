Workflow

Run the project step-by-step (OS independent)

This guide starts from the beginning: install Python, install the required tools, run the pipeline, and find your results. No prior bioinformatics experience needed.

Pipeline schematic

Think of the pipeline as an assembly line: each step produces a file that the next step uses.

Input
blaZ_filtradas.fasta
→
MUSCLE
Alignment
*_aligned.fasta
→
ClipKIT
Trimming
*_aligned_trimmed.fasta
→
IQ-TREE 2
Tree
iqtree_results/

1. Install Python (required)

Install Python 3 from the official website, then confirm it works from a terminal.

  • Download: python.org/downloads
  • Verify:
python --version

If this command fails on your computer, install Python again and make sure it is added to your PATH.

2. Get the project folder ready

You should have a folder that contains code/ and your input FASTA file. In this workspace the input file is blaZ_filtradas.fasta.

Open a terminal

Windows / macOS / Linux

Use PowerShell, Terminal, or your preferred shell.

Go to the project folder

Change directory

Navigate to the folder where this repo is located.

Confirm you are in the right place

Look for code/

The pipeline script is code/aln_trim_tree.py.

3. Install the external tools

The pipeline calls three programs. You may already have some of them in this repository (Windows binaries), but on other operating systems you will download them yourself.

MUSCLE

Needed for alignment.

Download MUSCLE

ClipKIT

Needed for trimming. Install it so you can run clipkit.

Download ClipKIT

IQ-TREE 2

Needed for the tree.

Download IQ-TREE 2

Tip: if a program is not on your PATH, you can still run the pipeline by passing explicit paths using --muscle-exe, --clipkit-exe, and --iqtree-exe.

4. Run the pipeline

From the repository root (the folder that contains code/), run:

python .\code\aln_trim_tree.py --input .\blaZ_filtradas.fasta

Optional examples

Choose an outgroup or change ClipKIT trimming mode:

python .\code\aln_trim_tree.py --input .\blaZ_filtradas.fasta --outgroup SAU40
python .\code\aln_trim_tree.py --input .\blaZ_filtradas.fasta --clipkit-mode smart-gap

5. Find your results

  • blaZ_filtradas_aligned.fasta
  • blaZ_filtradas_aligned_trimmed.fasta
  • iqtree_results/ (tree files, logs)

The main tree file is usually *.treefile inside iqtree_results/.

Troubleshooting (common issues)

Python not found

Reinstall Python and make sure it is added to your PATH, then retry python --version.

“clipkit: command not found”

Install ClipKIT, or pass its full path with --clipkit-exe.

“No such file”

Make sure blaZ_filtradas.fasta exists and you are running the command from the repo root.
